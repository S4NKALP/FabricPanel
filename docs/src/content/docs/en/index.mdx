## What is Tsumiki?

Tsumiki (formerly Hydepanel) is a modular status bar for the Hyprland Wayland compositor.
It is built on the Fabric widget system and designed around small, composable widgets.

## Features

  <Card title="Tailored for Hyprland" icon="seti:vala">
