## What is Tsumiki?

Tsumiki (formerly Hydepanel) is a modular status bar for the Hyprland Wayland compositor.
It is built on Fabric and designed for users who want a customizable but lightweight panel.

## What You Get

- Hyprland-first behavior and event integration.
- Modular widgets and optional modules.
- SCSS-based theming and configurable layout.
- Built-in notifications, OSD, dock, launcher, and quick settings support.

## Recommended Path

1. Install Tsumiki from [Installation](/en/getting-started/installation).
2. Configure layout and widgets in [Configuration](/en/configuring/config).
3. Customize colors and style in [Making Themes](/en/theming/making-themes).
4. Review [Post Installation](/en/resources/post-install) for Hyprland layer rules.
