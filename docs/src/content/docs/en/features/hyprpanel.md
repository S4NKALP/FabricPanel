Tsumiki is a Hyprland-first panel built with modular widgets and themeable styling.

## Why It Fits Hyprland

- Modular widgets you can place by section.
- Native support for common Hyprland workflows.
- Lightweight runtime for daily desktop use.
- Flexible theming through SCSS and `theme.toml`.

## Core Modules

- Bar
- Notifications
- Dock
- Overview
- On-screen display (OSD)

## Typical Widgets

- Workspaces
- Window title
- Date and time
- Battery, volume, brightness
- System tray and media controls
